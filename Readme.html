Food App Design Using React Native.

 This food app react native template is created for React Native developers. React native developers can easily use this food app template to build food related ecommerce apps.

 Note : This template only has routing and UI design code.

Major Pages included:

  • Login and Registration screens.
  • Individual screens merged into tabs for user point of view.
  • Tabs are like Home, Cart, Account etc.
  • Buttons available to add/remove elements  in cart page.

 

Features:

  • Ready to integrate services to make all food items as dynamic.
  • Easy to deploy on app store and play store
  • Better navigation between screens , handled by react-navigation
  • Easy to style by your own similar to CSS with light modifications in properties.

Installation Instructions:

Follow these instructions to setup food template react native app

  • Extract the project "foodie.zip".
  • Open command prompt terminal.
  • Enter into project root directory.
  • Type "npm install", to install all required npm modules.

Since, this project is for react-native developers, it is assumed that you have an Android or iOS emulator running in your development machine.

 

Node module Customizations:

If some of the package modules producing warning due to old version of it, Kindly update respective files from "npmModuleCustomize"


How to run:

  •  Android

            run this command "react-native run-android"

  • iOS

            run this command "react-native run-ios"
